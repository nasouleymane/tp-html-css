# TP HTML/CSS

## Description
Mini projet HTML/CSS réalisé dans le cadre d'un TP encadré.
Le site contient une page d'accueil, un formulaire complet et une page de confirmation.

## Structure du projet
tp-html-css/
├── index.html
├── pages/
│   ├── formulaire.html
│   └── confirmation.html
└── assets/
    └── css/
        └── style.css

## Comment exécuter
1. Placer le dossier dans htdocs de XAMPP
2. Ouvrir le navigateur et aller sur localhost/dev-web-2025-2026/tp-html-css/

## Choix réalisés
- Palette de couleurs : bleu (#2563eb), blanc cassé (#f8fafc), gris foncé (#1e293b)
- Un seul fichier CSS pour tout le site
- Pas de framework CSS, pas de JS
